# Expense Tracker

A Single Page Application (SPA) for tracking personal expenses. Built with vanilla JavaScript, HTML, and CSS, communicating with a Node.js backend.

## Features
- **Add/Remove Expenses**: Manage a list of expenses with amount and due dates.
- **Server Sync**: Persists state via RESTful API calls (GET/POST).
- **Responsive Design**: Styled with Flexbox for layout.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js

## Usage
Start the server and navigate to the localhost URL.
